# ReSale Agent 路线图

## 当前判断

V1 已经跑通主链路：图片识别、用户确认、动态追问、本地相似商品、规则估价、出售文案、Markdown 导出和模拟议价。V2 先围绕“更可解释、更可复用的出售辅助”做小步增强，不继续扩平台功能。

## V1 收口清单

- 保持三类商品范围：数码产品、图书、小家电。
- 保持本地 SQLite 模拟商品库，不接入真实二手平台。
- 保持估价由规则计算，大模型只做识别和文本生成辅助。
- 补齐新环境启动、CI、固定演示案例和 README 说明。
- 所有新增能力必须能在没有模型密钥时安全降级。

## V2 已完成增量

- 增强估价解释：展示规则估价、市场中位价、样本数量和融合权重。
- 增加轻量草稿管理：保存历史会话、重新打开草稿、删除草稿。
- 增加多平台风格文案：生成闲鱼、转转、小红书风格文案，但不触发真实发布。
- 增加成交反馈记录：保存最终成交价、成交渠道和估价偏差。
- 增加成交反馈汇总视图：展示成交数、分类命中比例和最近成交偏差，并支持按类别/渠道筛选。

## V2 候选方向

- 支持更多商品类别，例如服装、鞋包、家具。
- 增加图片相似度检索，但仍不爬取真实平台数据。

## V3 候选方向

- 接入合规价格数据源，替换或增强本地模拟数据。
- 管理多件闲置物品，形成个人闲置库存。
- 根据发布后的浏览量、收藏量和咨询量动态调整建议价。
- 在明确授权和合规边界内适配真实平台发布流程。

## 不做事项

- V1 不做真实平台登录、自动发布、真实买家聊天、支付、订单、物流和售后。
- V1 不做微服务、消息队列、Kubernetes 或复杂权限系统。
- V1 不把价格交给大模型直接决定。
