# 三分钟演示脚本

## 0:00 - 0:30 项目介绍

ReSale Agent 是一个 AI 二手物品出售助手。它不是二手交易平台，而是帮助用户从一张闲置物品照片快速得到可发布的出售方案。

## 0:30 - 1:10 上传与识别

选择“数码产品”，上传 Keychron K2 机械键盘图片。系统返回商品类型、品牌、型号、颜色、外观状态、可见瑕疵和置信度。强调所有字段都可以修改，最终以用户确认为准。

## 1:10 - 1:45 动态追问

确认识别结果后，系统根据数码产品规则追问原价、购买时间、功能状态、维修记录、配件和额外瑕疵。补充完整后，系统停止追问。

## 1:45 - 2:20 估价与文案

生成出售方案。说明估价不是大模型随口生成，而是由折旧、成色、配件、维修记录和本地相似商品中位数融合得到。展示挂牌价、成交区间、最低接受价、估价依据、商品标题、描述、关键词和拍照建议。

## 2:20 - 2:45 模拟议价

输入“150 包邮行不行”。系统识别为砍价，如果低于最低接受价，会礼貌拒绝并给出可继续沟通的价格边界。

## 2:45 - 3:00 总结

总结项目亮点：多阶段 Agent、结构化状态、用户确认、规则估价、价格边界校验、模型不可用时可降级、核心逻辑可单元测试。
